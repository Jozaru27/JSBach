# JSBach  
Configuración avanzada de router con sistema en Bash y Apache  

Gestión de IP, enrutamiento, bridge, firewall, DMZ, switch,...

---

## WAN

- Configuración de rutas estáticas, puertas de enlace, máscara y DNS locales.

---

## ENRUTAMIENTO

- Permite que los dispositivos conectados a JSBach tengan acceso a internet.

## BRIDGE

- Permite acceso a los servicios del Switch
- Configuración de Vlan
- Modificación TAG/UNTAG
- Aislamiento (EBTABLES)

---

## FIREWALL

- Permite aislar y desconectar Vlan
- Gestionar puertos en whitelist
- Restringir accesos mediante IP/MAC
- Cortar accesos a interfaces / Vlan

---

## DMZ

- Permite configurar DMZ en Puertos, según su IP y protocolo

---

## SWITCH

- Gestión de Switches
- Restricciones de MAC mediante ACL
- Privilegios de MAC mediante ACL
